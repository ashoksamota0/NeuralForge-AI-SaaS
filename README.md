# NeuralForge AI – Full Stack AI SaaS Platform

NeuralForge AI is a full-stack AI-powered SaaS platform that enables users to generate content, create blog titles, review resumes, generate AI images, remove image backgrounds, and perform intelligent image editing through an intuitive web interface. The platform combines modern AI services with secure authentication and cloud-based media management to deliver a complete productivity solution.

## ✨ Features

## 🔧 Installation

### Clone Repository

```bash
git clone https://github.com/ashoksamota0/NeuralForge-AI-SaaS.git
```

### Install Dependencies

Frontend:

```bash
cd client
npm install
```

Backend:

```bash
cd server
npm install
```

### Environment Variables

Create `.env` files for both client and server and configure:

- Clerk API Keys
- Gemini API Key
- Cloudinary Credentials
- ClipDrop API Key
- Database Connection URL

### Run the Application

Backend:

```bash
npm start
```

Frontend:

```bash
npm run dev
```
